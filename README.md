## 功能：
  - 1.支持歌词字同步动画，同步歌词进度 （可兼容其它音乐软件的歌词文件）歌词字同步算法
  - 2.支持多歌手写真动画交叉轮播
  - 3.音频动态频谱动画
  - 4.基本的音乐播放器功能（无需解释，该有的肯定有）
  - 5.基本的歌曲导入（选择导入，本地全盘扫描导入，支持mp3,wav,flac音乐文件(仅扫描带有歌手歌曲名文件名的文件,格式：歌手名 - 歌曲名.后缀名)）
- 注意：性能或系统版本较低，可能会影响运行时的性能(些许影响)，推荐WIn11，Win11有特别优化[doge]
